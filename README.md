변수
* 메모리(RAM)에 값을 저장하기 위해 할당하는 공간
* 할당 후 내부의 값 변경가능

상수
* 메모리(RAM)에 값을 저장하기 위해 할당하는 공간
* 할당 후 내부의 값 변경 불가

변수 이름과 값 객체
